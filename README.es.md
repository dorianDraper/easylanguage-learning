## 📘 Descripción general

Este repositorio contiene notas, ejemplos y código desarrollado durante el estudio de **EasyLanguage**, el lenguaje de programación utilizado en plataformas de trading como *TradeStation* y *MultiCharts*.

El contenido está organizado en dos rutas de aprendizaje estructuradas:

- **01 – Fundamentals**: Conceptos básicos del lenguaje y componentes de trading elementales
- **02 – Advanced**: Desarrollo de estrategias, diseño modular, sistemas multi-data y gestión del riesgo

El objetivo es **documentar el proceso de aprendizaje** mientras se construyen progresivamente **componentes reutilizables para sistemas de trading algorítmico**.

> ℹ️ *Todas las estrategias e indicadores están documentados en inglés y español.*

> ⚠️ **El contenido de este repositorio representa ideas de diseño y experimentos de aprendizaje — no estrategias de trading validadas.** No se ha realizado backtesting sistemático ni optimización de parámetros sobre ninguno de los sistemas aquí documentados. El backtesting, el análisis walk-forward y la evaluación de rendimiento se gestionan en un repositorio separado.

## 🧱 01 – Fundamentals

Esta sección cubre los bloques de construcción esenciales para desarrollar **indicadores** y **sistemas de trading básicos** en EasyLanguage.

### Contenido

- Sintaxis y estructura de EasyLanguage
- Variables, inputs y tipos de datos
- Funciones integradas y palabras reservadas
- Indicadores y osciladores
- Estudios ShowMe
- Estudios PaintBar
- Estrategias de trading básicas
- Herramientas de análisis de gráficos

🎯 **Objetivo:** Comprender cómo se expresa la lógica de trading de forma programática en EasyLanguage y construir una base sólida para el desarrollo de sistemas más complejos.

---

## 🚀 02 – Advanced

Esta sección se centra en el **trading sistemático** y el **desarrollo profesional de estrategias**, incluyendo arquitecturas multi-data que incorporan relaciones intermercado.

### Contenido

- Diseño y arquitectura de estrategias
- Diseño de código modular y reutilizable
- Estrategias multi-data con filtros intermercado
- Indicadores avanzados y análisis de microestructura
- Componentes de infraestructura (gestión del riesgo, sizing de posición, control de sesión)
- Abstracción de código y mantenibilidad

🎯 **Objetivo:** Pasar de scripts simples a **sistemas de trading robustos, escalables y mantenibles** que puedan servir como base para un portfolio de trading sistemático.

Este repositorio está concebido estrictamente con **fines educativos**.

- El código y los ejemplos pueden usarse y adaptarse libremente para el aprendizaje personal.
- Algunos scripts representan **experimentos o trabajos en curso** mientras se exploran y refinan conceptos. No deben tratarse como sistemas terminados o listos para producción.
- **Nada en este repositorio constituye asesoramiento financiero ni una recomendación para operar ningún instrumento o estrategia específica.**
- Las estrategias e indicadores documentados aquí son exploraciones de diseño. No han sido sometidos a backtesting sistemático, validación walk-forward ni optimización. La lógica de diseño pasada no implica rendimiento de trading futuro.
